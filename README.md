# ⚡ Smart Electronics — Веб-сайт

Профессиональный интернет-магазин электронных компонентов и запчастей.

## 🚀 Запуск проекта

### 1. Backend (Node.js + Express)

```bash
cd backend
npm install
npm start
```

Сервер запустится на: `http://localhost:5000`

### 2. Frontend (React + Vite + Tailwind)

```bash
cd frontend
npm install
npm run dev
```

Сайт откроется на: `http://localhost:3000`
